This is the API

getApiAll.json

Return a list of all products. Each of these can be used for the other API endpoints.

URL

https://endoflife.date/api/all.json

HTTP Method

GET

Response Type

oas_any_type_not_mapped

Parameters

getApiProduct.json

Get EoL dates of all cycles of a given product.

URL

https://endoflife.date/api/{product}.json

HTTP Method

GET

Response Type

oas_any_type_not_mapped

Parameters

  • Path: product oas_any_type_not_mapped

    Product URL as per the canonical URL on the endofife.date website

getApiProductCycle.json

Gets details of a single cycle

URL

https://endoflife.date/api/{product}/{cycle}.json

HTTP Method

GET

Response Type

cycle

Parameters

  • Path: product oas_any_type_not_mapped

    Product URL as per the canonical URL on the endofife.date website

  • Path: cycle oas_any_type_not_mapped

    Release Cycle for which the details must be fetched
