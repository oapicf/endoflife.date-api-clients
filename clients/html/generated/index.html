get /api/{product}/{cycle}.json
Single cycle details (getApiProductCycle.json)
Gets details of a single cycle

Path parameters

product (required)
Path Parameter — Product URL as per the canonical URL on the endofife.date website default: null
cycle (required)
Path Parameter — Release Cycle for which the details must be fetched default: null

Return type

cycle

  "eol" : "",
  "releaseDate" : "",
  "link" : "",
  "lts" : "",
  "discontinued" : "",
  "cycle" : "",
  "support" : "",

}

  1. cycle - Cycle

cycle - Cycle Up

Details of a single release cycle of a given product. There might be some slight variations to this depending on the product.
cycle (optional)
oas_any_type_not_mapped Release Cycle
releaseDate (optional)
oas_any_type_not_mapped Release Date for the first release in this cycle format: date
eol (optional)
oas_any_type_not_mapped End of Life Date for this release cycle
latest (optional)
oas_any_type_not_mapped Latest release in this cycle
link (optional)
oas_any_type_not_mapped Link to changelog for the latest release, if available
lts (optional)
oas_any_type_not_mapped Whether this release cycle has long-term-support (LTS). Can be a date instead in YYYY-MM-DD format as well if the release enters LTS status on a given date.
support (optional)
oas_any_type_not_mapped Whether this release cycle has active support format: date
discontinued (optional)
oas_any_type_not_mapped Whether this cycle is now discontinued. format: date
